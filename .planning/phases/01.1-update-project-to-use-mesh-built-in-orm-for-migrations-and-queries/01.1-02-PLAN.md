Key ORM API patterns (all verified working in reference project):

```mesh
# Basic insert — returns Map<String, String> with generated values (including UUID id)
let fields = %{"email" => email, "password_hash" => password_hash}
let row = Repo.insert(pool, User.__table__(), fields)?

# Get by primary key
let row = Repo.get(pool, User.__table__(), id)?

# Get by non-PK column
let row = Repo.get_by(pool, Session.__table__(), "token", token_value)?

# Query builder with where + order + select
let q = Query.from(User.__table__())
  |> Query.where(:email, email)
  |> Query.where_raw("password_hash = crypt(?, password_hash)", [password])
  |> Query.select_raw(["id::text", "email", "created_at::text"])
let rows = Repo.all(pool, q)?

# Update where
let q = Query.from(Session.__table__())
  |> Query.where(:token, token)
Repo.update_where(pool, Session.__table__(), %{"field" => "value"}, q)?

# Delete where
let q = Query.from(Session.__table__())
  |> Query.where(:token, token)
Repo.delete_where(pool, Session.__table__(), q)

# Delete by PK
Repo.delete(pool, OrgMembership.__table__(), id)?

# Raw SQL for complex queries (crypt, NOW(), subqueries)
let rows = Repo.query_raw(pool, "SELECT crypt($1, gen_salt('bf', 12)) AS hash", [password])?
Repo.execute_raw(pool, "UPDATE ... SET ... = NOW() WHERE ...", [...])?

# Two-step pattern for server-side functions:
# Step 1: Get computed value
let hash_rows = Repo.query_raw(pool, "SELECT crypt($1, gen_salt('bf', 12)) AS hash", [password])?
let password_hash = Map.get(List.head(hash_rows), "hash")
# Step 2: ORM insert with literal value
let fields = %{"email" => email, "password_hash" => password_hash}
let row = Repo.insert(pool, User.__table__(), fields)?

# Two-step pattern for NOW() updates:
let ts_rows = Repo.query_raw(pool, "SELECT now()::text AS ts", [])?
let ts = Map.get(List.head(ts_rows), "ts")
Repo.update_where(pool, Table.__table__(), %{"revoked_at" => ts}, q)?

# UUID comparisons need ::uuid cast
Query.where_raw("id = ?::uuid", [id])
Query.where_raw("org_id = ?::uuid", [org_id])
```

  <name>Task 1: Create centralized storage/queries.mpl with all ORM query functions</name>
  <files>src/storage/queries.mpl</files>
  <action>
Create `src/storage/queries.mpl` (create the `src/storage/` directory if needed). This module provides ALL data access for the application. Handlers will import and call these functions instead of writing inline SQL.

Import all model types at the top:
```mesh
from Src.Types.User import User, OrgMembership, Session, Invite, PasswordResetToken
from Src.Types.Project import Organization, Project, ApiKey
```

All functions take `pool :: PoolHandle` as first argument and return `Type!String` (Result with String error). Use `?` operator throughout.

**User functions:**

1. `pub fn create_user(pool :: PoolHandle, email :: String, password :: String) -> String!String` — Two-step: Repo.query_raw for `SELECT crypt($1, gen_salt('bf', 12)) AS hash` to get password_hash, then Repo.insert with %{"email" => email, "password_hash" => password_hash}. Return the id from the inserted row: `Map.get(row, "id")`.

2. `pub fn authenticate_user(pool :: PoolHandle, email :: String, password :: String) -> Map<String, String>!String` — Query.from(User.__table__()) |> Query.where(:email, email) |> Query.where_raw("password_hash = crypt(?, password_hash)", [password]) |> Query.select_raw(["id::text", "email"]). Return first row or Err("not found").

3. `pub fn get_user_by_email(pool :: PoolHandle, email :: String) -> Map<String, String>!String` — Query.from + Query.where(:email, email) + select_raw(["id::text", "email"]). Return first row or Err("not found").

4. `pub fn update_user_password(pool :: PoolHandle, user_id :: String, new_password :: String) -> Int!String` — Two-step: Repo.query_raw for crypt hash, then Repo.update_where with %{"password_hash" => hash} where id = ?::uuid.

**Session functions:**

5. `pub fn create_session(pool :: PoolHandle, user_id :: String) -> String!String` — Generate token as two UUID4s with hyphens stripped concatenated (matching reference project: 64 hex chars). Repo.insert with %{"token" => token, "user_id" => user_id}. Return token.

6. `pub fn validate_session(pool :: PoolHandle, token :: String) -> Map<String, String>!String` — Query.from(Session.__table__()) |> Query.where(:token, token) |> Query.where_raw("expires_at > now()", []) |> Query.select_raw(["token", "user_id::text"]). Return first row or Err("not found").

7. `pub fn delete_session(pool :: PoolHandle, token :: String) -> Int!String` — Repo.delete_where with Query.where(:token, token).

8. `pub fn delete_user_sessions(pool :: PoolHandle, user_id :: String) -> Int!String` — Repo.delete_where with Query.where_raw("user_id = ?::uuid", [user_id]).

9. `pub fn cleanup_expired_sessions(pool :: PoolHandle) -> Int!String` — Repo.delete_where with Query.where_raw("expires_at < now()", []).

**Organization functions:**

10. `pub fn insert_org(pool :: PoolHandle, name :: String, slug :: String) -> String!String` — Repo.insert with %{"name" => name, "slug" => slug}. Return id.

11. `pub fn get_org(pool :: PoolHandle, id :: String) -> Map<String, String>!String` — Query.from + Query.where_raw("id = ?::uuid", [id]) + select_raw(["id::text", "name", "slug", "created_at::text"]). Return first row or Err("not found").

12. `pub fn list_user_orgs(pool :: PoolHandle, user_id :: String) -> List<Map<String, String>>!String` — Query.from(Organization.__table__()) |> Query.join_as(:inner, OrgMembership.__table__(), "m", "m.org_id = organizations.id") |> Query.where_raw("m.user_id = ?::uuid", [user_id]) |> Query.select_raw(["organizations.id::text", "organizations.name", "organizations.slug", "organizations.created_at::text", "m.role"]) |> Query.order_by_raw("organizations.created_at").

**Membership functions:**

13. `pub fn add_member(pool :: PoolHandle, org_id :: String, user_id :: String, role :: String) -> String!String` — Repo.insert with %{"org_id" => org_id, "user_id" => user_id, "role" => role}. Return id.

14. `pub fn check_membership(pool :: PoolHandle, org_id :: String, user_id :: String) -> Map<String, String>!String` — Query.from(OrgMembership.__table__()) |> Query.where_raw("org_id = ?::uuid AND user_id = ?::uuid", [org_id, user_id]) |> Query.select_raw(["id::text", "role"]). Return first row or Err("not a member").

**Invite functions:**

15. `pub fn create_invite(pool :: PoolHandle, org_id :: String, email :: String, token :: String, invited_by :: String) -> String!String` — Repo.insert. Use Repo.execute_raw for the `expires_at` since it needs `now() + interval '7 days'` which Repo.insert can't express. Alternative: use Repo.query_raw to get the expiry timestamp first, then Repo.insert with the literal string. Return id.

16. `pub fn get_pending_invite(pool :: PoolHandle, token :: String) -> Map<String, String>!String` — Query.from(Invite.__table__()) |> Query.where(:token, token) |> Query.where_raw("accepted_at IS NULL AND revoked_at IS NULL AND expires_at > now()", []) |> Query.select_raw(["id::text", "org_id::text", "email"]). Return first row or Err("not found").

17. `pub fn check_pending_invite_by_email(pool :: PoolHandle, org_id :: String, email :: String) -> Bool!String` — Query.from + where_raw for org_id/email + where_raw for accepted/revoked/expires. Return List.length(rows) > 0.

18. `pub fn check_existing_member_by_email(pool :: PoolHandle, org_id :: String, email :: String) -> Bool!String` — Query.from(OrgMembership.__table__()) join users on user_id, where org_id and users.email match. Return List.length(rows) > 0.

19. `pub fn accept_invite(pool :: PoolHandle, invite_id :: String) -> Int!String` — Two-step: get now() timestamp, then Repo.update_where with %{"accepted_at" => ts}.

20. `pub fn revoke_invite(pool :: PoolHandle, invite_id :: String, org_id :: String) -> Int!String` — Two-step: get now() timestamp, then Repo.update_where with %{"revoked_at" => ts} where id = ?::uuid AND org_id = ?::uuid AND revoked_at IS NULL.

21. `pub fn list_invites(pool :: PoolHandle, org_id :: String) -> List<Map<String, String>>!String` — Query.from + where_raw for org_id + select_raw all fields + order_by created_at DESC.

**Password Reset functions:**

22. `pub fn invalidate_existing_reset_tokens(pool :: PoolHandle, user_id :: String) -> Int!String` — Two-step: get now(), then Repo.update_where with %{"used_at" => ts} where user_id = ?::uuid AND used_at IS NULL.

23. `pub fn create_reset_token(pool :: PoolHandle, user_id :: String, token_hash :: String) -> String!String` — Two-step: get now() + interval '1 hour' timestamp via Repo.query_raw("SELECT (now() + interval '1 hour')::text AS ts", []), then Repo.insert with %{"user_id" => user_id, "token_hash" => token_hash, "expires_at" => ts}. Return id.

24. `pub fn validate_reset_token(pool :: PoolHandle, token_hash :: String) -> Map<String, String>!String` — Query.from + where(:token_hash, token_hash) + where_raw("expires_at > now() AND used_at IS NULL", []) + select_raw(["id::text", "user_id::text"]). Return first row or Err("not found").

25. `pub fn mark_reset_token_used(pool :: PoolHandle, token_id :: String) -> Int!String` — Two-step: get now(), then update_where with %{"used_at" => ts}.

**Project functions:**

26. `pub fn create_project(pool :: PoolHandle, org_id :: String, name :: String) -> String!String` — Repo.insert with %{"org_id" => org_id, "name" => name}. Return id.

27. `pub fn list_projects(pool :: PoolHandle, org_id :: String) -> List<Map<String, String>>!String` — Query.from + where_raw("org_id = ?::uuid") + select_raw + order_by created_at.

**API Key functions:**

28. `pub fn create_api_key(pool :: PoolHandle, project_id :: String, key_hash :: String, key_prefix :: String, label :: String) -> String!String` — Repo.insert with all fields. Return id.

29. `pub fn list_api_keys(pool :: PoolHandle, project_id :: String) -> List<Map<String, String>>!String` — Query.from + where_raw("project_id = ?::uuid") + select_raw + order_by created_at DESC.

30. `pub fn revoke_api_key(pool :: PoolHandle, key_id :: String) -> Int!String` — Two-step: get now(), then update_where with %{"revoked_at" => ts} where id = ?::uuid AND revoked_at IS NULL.

**OAuth-specific functions:**

31. `pub fn store_oauth_state(pool :: PoolHandle, state :: String) -> Int!String` — Insert a session row with token = "oauth_state_" <> state, user_id as a placeholder UUID, short expiry. Use Repo.query_raw("SELECT (now() + interval '10 minutes')::text AS ts", []) for expiry, then Repo.insert.

32. `pub fn validate_oauth_state(pool :: PoolHandle, state :: String) -> Bool!String` — Query.from(Session.__table__()) where token = "oauth_state_" <> state AND expires_at > now(). Return List.length(rows) > 0.

33. `pub fn delete_oauth_state(pool :: PoolHandle, state :: String) -> Int!String` — Repo.delete_where where token = "oauth_state_" <> state.

34. `pub fn upsert_oauth_user(pool :: PoolHandle, email :: String) -> String!String` — Check if user exists by email. If yes, return id. If no, create with random password (crypt of uuid4). Return id.

All functions MUST use `pub fn` for cross-module access. All MUST use `?` operator for error propagation. Maintain bottom-up ordering: helper functions first, then public functions that call them.

IMPORTANT: For functions returning a single row from Repo.all, always check `List.length(rows) > 0` and return Err if empty. Never assume rows exist.
